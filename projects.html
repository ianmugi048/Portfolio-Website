My Projects

Here are some of the projects I've worked on, showcasing my skills and knowledge in cybersecurity and programming.

  • Project 1: Cybersecurity Awareness Campaign

    • This project involved creating a comprehensive cybersecurity awareness campaign to educate friends, family and classmates about the importance of cybersecurity and best practices to stay safe online.

    The campaign included informative posters, social media posts, and talks to raise awareness about common cyber threats and how to mitigate them.

    The goal was to empower individuals with knowledge and tools to protect themselves from cyber threats.

  • Project 2: Personal Portfolio Website

    • This project involved creating a personal portfolio website to showcase my skills, projects, and certifications in the field of cybersecurity.

    The website was built using HTML and CSS, and it serves as a platform to share my journey and connect with others in the cybersecurity community.

    The website includes sections for my projects, certifications, and a contact form for potential collaborations.

  • Project 2: Wazuh SIEM Agent Deployment

    • This project involved deploying Wazuh SIEM agents on my local systems to monitor and analyze security events in real-time.

    The deployment included configuring the agents, setting up alerts, and analyzing logs to identify potential security threats.

    The goal was to enhance the security posture of the systems and ensure compliance with security policies.

  • Project 3: Wireshark Packet Analysis

    • This project involved capturing and analyzing network packets using Wireshark to understand network traffic and identify potential security threats.

    The analysis included filtering packets, examining protocols, and identifying anomalies in network traffic.

    The goal was to gain insights into network behavior and enhance my skills in network analysis.
